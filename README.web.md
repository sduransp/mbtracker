# MBTracker Web UI

This folder contains a minimal Vite-based frontend for MBTracker.

## Development

```bash
npm run dev
```
- API: http://127.0.0.1:5174
- Vite dev server: http://127.0.0.1:5173

## Build

```bash
npm run build
```
Outputs static files into `web/dist/` which are served by the Express server.

## Notes
- Keep UI simple and responsive.
- Avoid heavy dependencies.
